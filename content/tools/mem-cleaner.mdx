### Getting Started with RepoVox
Follow these simple steps to set up **RepoVox** and automate your content sharing:

<Steps>

### Login & Connect GitHub
Sign in to **RepoVox** and link your GitHub account to start integrating your repositories.
 
<img
  className="mt-4 overflow-hidden rounded-lg border block dark:hidden"
  src="/_static/docs/lite-repo-connect.png"
  alt="Light Mode"
/>

<img
  className="mt-4 overflow-hidden rounded-lg border hidden dark:block"
  src="/_static/docs/repo-connect.png"
  alt="Dark Mode"
/>

### Install RepoVox GitHub App
To enable webhook connectivity, install the **RepoVox GitHub App** on your GitHub account or specific repositories.
 
![RepoVox App Installation](/_static/docs/repovox-app-installation.gif)

### Integrate Social Media
Connect your preferred social media platforms like **Twitter, LinkedIn, and Dev.to** to automate content posting.

<img
  className="mt-4 overflow-hidden rounded-lg border block dark:hidden"
  src="/_static/docs/lite-integration-page.png"

/>

<img
  className="mt-4 overflow-hidden rounded-lg border hidden dark:block"
  src="/_static/docs/integration-page.png"
  alt="Dark Mode"
/>

### Configure Your Repo & Actions
- Select a **GitHub repository** and **branch**.
- Choose the **trigger action** (e.g., Pull Request Merged, New Commit, etc.).
- Define where the content should be posted.

That's it! **RepoVox** will now generate AI-powered content based on your GitHub activity and post it automatically.
